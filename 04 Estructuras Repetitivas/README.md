# Práctico 4: Estructuras repetitivas

## Contenido 

El trabajo práctico incluye :

1. **Contador del 0 al 100**  
2. **Cantidad de dígitos de un número**  
3. **Suma entre dos números (excluyentes)**  
4. **Suma de números ingresados hasta 0**  
5. **Juego: Adivina el número**  
6. **Números pares del 100 al 0**  
7. **Suma del 0 a un número positivo**  
8. **Clasificación de 100 números**  
9. **Media de 100 números**  
10. **Inversión de dígitos**
